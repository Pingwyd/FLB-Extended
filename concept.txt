Admin Privilege Levels (Principle of Least Privilege):

1. MODERATOR (Content Management):
   - Delete/hide marketplace listings (Real Estate & Workers)
   - Review and act on reported content
   - Mark listings as fraudulent/spam
   - View user reports and flagged content
   - CANNOT ban users
   - CANNOT access payment/wallet data
   - CANNOT modify user accounts
   - CANNOT manage other moderators/admins
   Authentication: Email + Password

2. ADMIN (User Management):
   - Ban/unban users
   - Review ban appeals
   - Approve/reject verification documents (NIN, Passport, etc.)
   - View user activity logs
   - Manage user account status
   - CANNOT delete listings (must report to moderators)
   - CANNOT delete other admins
   - CANNOT access financial transaction details
   - CANNOT modify system configuration
   Authentication: Email + Password + OTP (SMS/Email)

3. SUPER ADMIN (Full System Access):
   - All moderator privileges
   - All admin privileges
   - Create/delete moderators and admins
   - Access financial reports and transaction logs
   - System configuration and settings
   - Database management
   - Audit log access
   - Emergency system controls
   Authentication: Email + Password + OTP + IP Whitelist + Security Questions

Audit Logging:
- All admin actions logged with timestamp, IP, and admin ID
- Deletion actions require reason/notes
- Ban actions require evidence and reason
- All logs immutable and accessible only to Super Admin
